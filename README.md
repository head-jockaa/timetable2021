テレビ番組表の記録(2021年分)
==

概要
--

日本全国のテレビ番組表１年分をどれだけ小さな容量でアーカイブできるか実験をおこなうプロジェクトです。  
なおかつ、まとまったデータを欲しい人が大量にHTTPリクエストを投げなくても済むようなアーカイブ作成も目的としています。  
2021年1月18日分から `timetable.yanbe.net` から離れて独自にデータ収集を始めており、うp主によっぽどの惨事が降りかからない限りは継続予定です。  
ある程度まとまってから更新をおこなうので、毎日更新ではない事に注意してください。

使い方
--

`https://github.com/head-jockaa/timetable2021` のページにある緑色の `Code` ボタンを押して `Download ZIP` を選んでください。  
ダウンロードされたZIPファイルを展開して `2021.html` をブラウザで開けばOKです。  
`scripts` フォルダの中にある３つの圧縮ファイル `timetables.js.7z`, `titles.js.7z`, `descriptions.js.7z` をそれぞれ展開する必要があります。  
7-Zip形式に対応した解凍ソフトを使ってください。

データ収集対象
--

* 全都道府県の地上波テレビ、BSデジタルテレビ
  * (ラジオ放送およびCS放送はとりあえず対象外)
* 当該サイトの番組表テーブル上に見えている情報すべて
  * 開始時刻、放映時間、ジャンル、各種アイコン、番組名、番組概要  
  * (番組名をクリックした時の詳細情報は対象外)

設計
--

* 辞書ファイルを用意したり、関東キー局との差分を求めることにより、データサイズをなるべく小さく抑える。  
* javascriptを利用し、番組表のデザインに自由度を与える。
